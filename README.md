## archmixesbot
archmixesbot


1. Set the following in `post_tweet.sh`:
- consumer_key
- consumer_secret
- access_token
- token_secret

2. Before running:
- `current_line.txt` should be set to 0
- `tweet_list.txt` should be empty 

3. Run with: `python watcher.py`
